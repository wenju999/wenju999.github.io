## Mac快捷键
```
显示隐藏文件：command + shift + .
选中区域截屏： command + shift + 4  
选中区域截屏到粘贴板：command + shift + control + 4
全屏截屏：command + shift + 3  
全屏截屏到粘贴板：command + shift + control + 3
```

## 基础通用
```
复制：command(control) + C
粘贴：command(control) + V
剪切：command(control) + X
全选：command(control) + A
部分查找：command(control) + F
全部查找：command(control) + shift + F
快速选择部分文件或文件夹：先“点击选中一个文件”，再command + shift + “另外选中文件” 
```
